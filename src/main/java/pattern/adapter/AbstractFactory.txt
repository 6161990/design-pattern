### [Adapter Pattern 의 핵심]
* 서로 다른 인터페이스를 중간에서 연결해주는 기능

### [Adapter Pattern 의 효용]
* 이미 사용중이거나 정의된 인터페이스들을 중간에서 맞춰서 적용해줄 수 있음
* 클라이언트에서 사용하던 방식대로 호출하여 사용할 수 있도록 조정해줄 수 있음
* 서로 안맞는 인터페이스를 변경하지 않고 중간에서 호출하여 사용할 수 있도록 제공함
* Adapter 를 사용함으로써 클라이언트가 사용하는 방식은 동일하면서 여러 기능이 제공될 수 있음

### [기타]
* 상속을 활용하여 구현하는 Adapter
* 객체 합성의 방법으로 구현하는 Adapter
